# CODSOFT Internship Tasks

This repository contains 3 projects completed for CODSOFT Data Science Internship.

## Projects

1. **Credit Card Fraud Detection**  
   - Detects fraudulent credit card transactions using Logistic Regression.
2. **TITANIC SURVIVAL PREDICTION**  
   - Predicts Titanic survival or movie ratings using ML models.
3. **Movie Rating Prediction**  
   - Predicts movie ratings based on features like director, genre, cast, etc.
