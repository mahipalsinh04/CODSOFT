# TITANIC SURVIVAL PREDICTION

## Overview
This project predicts Titanic survival or movie ratings using Machine Learning.

## Features
- Input: Passenger information (Age, Fare, Pclass, etc.) or movie features
- Output: Survival (0/1) or Movie Rating prediction
- Model used: Linear Regression / Random Forest
- Handles missing values and splits data into training and testing sets
